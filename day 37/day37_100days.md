### DAY 37 - STRING SLICING
String slicing is manipulating code for it to print specific parts of a string

```python
string = "Hello"
print(string[0:2])
```
This code will just print from H to l which means we have extracted some stuff from the string

```python
string = "Hello there , lay down your blasters"

print(string[:5])
print(string[6:14])
print(string[4:])

print(string[0:14:2])
print(string[::-1])

```
